Ejercicio Responsive Design (I)

Utilizando como base el CSS proporcionado, crea un HTML con el siguiente layout y comportamiento adaptativo.


1. Para anchuras de pantalla mayores de 900px:

El header ocupará 1 columna. El aside ocupará 2 columnas. Los articles ocuparán el espacio restante y se distribuirán 3+2.


2. Para anchuras de pantalla entre 600px y 900px:

El aside ocupará 3 columnas. Los articles ocuparán el espacio restante y se distribuirán 2+1+1+1.


3. Para anchuras de pantalla menores de 600px:

Todos los elementos ocuparán toda la anchura disponible.

CSS:

* {
  box-sizing: border-box;
}

header {
  background-color: lightpink;
  padding: 20px;
}

aside {
  background-color: lightblue;
  padding: 20px;
}

article {
  background-color: lightgreen;
  padding: 20px;
}

footer {
  background-color: darkgray;
  padding: 20px;
}
