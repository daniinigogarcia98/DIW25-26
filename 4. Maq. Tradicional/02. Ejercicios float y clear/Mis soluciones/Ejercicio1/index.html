El elemento div tiene dos propiedades que se pueden usar para alinear elementos: float y clear. La propiedad float se usa para alinear elementos horizontalmente. Cuando se usa, el elemento se coloca a la izquierda o a la derecha del contenedor, en función de la propiedad float que se aplica. La propiedad clear se usa para limpiar elementos. Cuando se usa, el elemento se elimina de la página, dejando espacio en blanco para otros elementos. En este ejercicio, se usa la propiedad float para alinear los elementos div1 y div2 a la derecha y a la izquierda, respectivamente. Para hacer que el elemento div1 se alinee a la derecha, se usa la propiedad float: right. Para hacer que el elemento div2 se alinee a la izquierda, se usa la propiedad float: left. Para limpiar los elementos, se usa la propiedad clear: both. Esta propiedad se aplica a un elemento hijo de un elemento div que tiene la propiedad clear establecida en both.
