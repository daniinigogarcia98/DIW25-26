Propiedades float y clear. Hack clearfix

Propiedad float

Usamos float para hacer que un elemento se desplace hacia la izquierda o la derecha dentro de su contenedor, permitiendo que el contenido siguiente lo rodee. Es muy útil, por ejemplo, para colocar imágenes a un lado del texto.

Tienes más información y ejemplos sobre float en W3Schools.
  • float: left: Hacemos que el elemento flote a la izquierda y el contenido siguiente lo rodea por la derecha.
  • float: right: Hacemos que el elemento flote a la derecha y el contenido siguiente lo rodea por la izquierda.
float: left

Este texto rodea la caja azul que flota a la izquierda. Vemos cómo el contenido se adapta y fluye alrededor del elemento flotante. Si añadimos más texto, seguirá rodeando la caja hasta que termine la altura de la caja flotante, momento en el que el texto continuará debajo de la caja. Puedes hacer más pequeña la ventana del navegador para ver cómo el texto se ajusta dinámicamente alrededor del elemento flotante.

float: right

Ahora la caja rosa flota a la derecha y el texto la rodea por la izquierda. Podemos usar float para crear diseños con columnas o para alinear imágenes y bloques de contenido.

Un aspecto importante acerca de la propiedad float es que los elementos flotantes no afectan el flujo del documento. Esto significa que el contenedor padre no se expande para incluir la altura de los elementos flotantes, lo que puede causar problemas de diseño. Para solucionar esto, podemos usar la propiedad clear. Si te fijas, este mismo párrafo se "montará" encima del elemento flotante justo anterior y se pegará al párrafo anterior porque el navegador no tendrá en cuenta al elemento flotante a la hora de calcular dónde debe ir.

Propiedad clear

Usamos clear para evitar que un elemento se sitúe al lado de elementos flotantes anteriores. Así conseguimos que el elemento baje por debajo de los elementos con float.

  • clear: left: El elemento baja por debajo de los elementos que flotan a la izquierda.
  • clear: right: El elemento baja por debajo de los elementos que flotan a la derecha.
  • clear: both: El elemento baja por debajo de todos los elementos flotantes, tanto a la izquierda como a la derecha.

El hack clearfix

Cuando usamos elementos flotantes (float), el contenedor padre no aumenta su altura para incluirlos, lo que puede romper el diseño. Para solucionarlo, usamos el hack clearfix, que fuerza al contenedor a "envolver" a los elementos flotantes.

El hack clearfix consiste en añadir una pseudo-clase ::after al contenedor, que limpia los floats automáticamente. Así, el contenedor recupera su altura original.

Ejemplo de CSS para clearfix:

.clearfix::after {
  content: "";
  display: table;
  clear: both;
}

¿Cómo lo usamos? Añadimos la clase clearfix al contenedor de los elementos flotantes:

<div class="clearfix">
  <div class="float-box float-left">float: left</div>
  <div class="float-box float-right">float: right</div>
</div>

Así conseguimos que el contenedor envuelva correctamente a los elementos flotantes y no se colapse.

Prueba a eliminar la clase clearfix del contenedor anterior, verás que este párrafo se sitúa en medio de los dos elementos flotantes.

Podrías pensar que la clase clearfix es innecesaria, dado que para conseguir el mismo efecto podemos usar clear: both en el párrafo siguiente a los elementos flotantes. Sin embargo, los efectos no son los mismos. Puedes hacer la prueba tú mismo. Observa como el margen superior no se comporta igual de una manera que de la otra.

El hack clearfix no es la única solución, pero sí la más efectiva. Otra solución conocida es usar la propiedad overflow en el contenedor padre, pero esto puede tener efectos secundarios indeseados como ocultar contenido desbordante, por eso es mejor usar clearfix.
