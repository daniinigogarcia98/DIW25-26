position: relative

El valor relative posiciona un elemento respecto a su posición normal en el flujo del navegador.

Se usan las propiedades top, right, bottom, left para especificar el desplazamiento del elemento respecto de su posición original. Estas propiedades aceptan valores negativos.

El viewport es el área visible en la pantalla de un dispositivo.

El valor absolute posiciona un elemento respecto a su ancestro posicionado no static más cercano. Si no encuentra ninguno, lo posiciona respecto al viewport*.

Un elemento con position: absolute no ocupa espacio en el flujo normal del documento.

(*) El viewport es el área disponible en la pantalla del navegador (sin contar la interfaz del navegador, como botones, barra de menú, etc.) es decir, es el área donde se mostrará la página web o el espacio para renderizar el documento web.
