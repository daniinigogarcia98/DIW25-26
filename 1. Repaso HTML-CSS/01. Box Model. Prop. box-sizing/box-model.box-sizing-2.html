Explicación:

Si eliminamos box-sizing: border-box, los elementos div tendrán un comportamiento diferente en cuanto al tamaño.

El ancho especificado del 50% solo incluirá el contenido, y el padding y el borde se añadirán a este ancho, haciendo que los div sean más anchos de ese 50%, lo que implica que DIV2 baje desborde y baje a la línea siguiente.
