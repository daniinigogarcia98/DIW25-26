La propiedad box-sizing

content-box
border-box

Explicación:

La propiedad CSS box-sizing determina cómo se calcula el tamaño total de un elemento HTML, incluyendo su contenido (content), relleno (padding) y borde (border).

Existen dos valores principales para box-sizing:

  • content-box: El tamaño total se calcula considerando únicamente el contenido interior del elemento, excluyendo el padding y el border.
  • border-box: El tamaño total se calcula incluyendo el contenido, el padding y el border, lo que hace que el tamaño total sea más predecible.

En el ejemplo, el div con content-box tiene un tamaño total de 330px, 300px por width, 10 px por cada uno de los paddings y 10px más por cada uno de los bordes.

Sin embargo, el div con border-box tiene un tamaño total de 300px, ya que el padding y el border se incluyen dentro de la medida especificada.
